# 1.1 什么是创新

# 1.1.1 创新的概念

起源于拉丁语, 原意: 更新、创造新的东西, 解决新的问题。

创新就是利用已有的自然资源或社会要素创造新的矛盾共同体的人类行为，或者是对旧有的一切所进行的替代、覆盖。

创新是指以现有的思维模式提出有别于常规或常人思路的见解为导向，利用现有的知识和物质，在特定的环境中，本着理想化需要或为满足社会需求，而改进或创造新的事物、方法、元素、路径、环境等，并能获得一定有益效果的行为。

特别强调“有益”二字。

# 1.1.2 创新的本质

--- 创新是创新思维蓝图的外化、物化、形式化。  
- 创新是以新思维、新发明和新描述为特征的一种概念化过程。

# 1.1.3 创新理论的发展过程

起源：

1912 年由哈佛大学熊彼特教授第一次把它引入到经济领域, 提出: 创新是指把一种新的生产要素和生产条件的 “新结合” 引入生产体系。包括: 引入一种新产品, 引入一种新的生产方法, 开

辟一个新的市场，获得原材料或半成品的一种新的供应来源。包含的范围很广，涉及到技术性变化的创新及非技术性变化的组织创新。

# 1.1.3 创新理论的发展过程

20 世纪 60 年代,新技术革命的迅猛发展。对“创新”的概念发展为 “技术创新”,把 “技术创新” 提高到 “创新” 的主导地位。

“技术创新是几种行为综合的结果，这些行为包括发明的选择、资本投入保证、组织建立、制定计划、招用工人和开辟市场等”。

# 1.1.3 创新理论的发展过程

80 年代,技术创新就是指新产品、新过程、新系统和新服务的首次商业性转化。

21 世纪, 信息技术推动下知识社会的形成及其对技术创新的影响进一步被认识, 科学界进一步反思对创新的认识: 技术创新是一个科技、经济一体化过程, 是技术进步与应用创新 “双螺旋结构” (创新双螺旋) 共同作用催生的产物。

创新的核心是“将新的理念、方法、技术或模式转化为实际价值的过程”，它必须同时具备以下3个关键属性：

价值性：能解决真实需求（如提高效率、降低成本、改善体验、创造新市场），而非无意义的“猎奇式新事物”；

突破性：相较于现有方案，存在本质差异（如技术原理突破、商业模式重构），而非简单的“换皮式修改”；

可行性与落地性：能被实际应用、推广或规模化，而非停留在“概念阶段”的空想。

只有当“新事物”同时满足“价值性”“突破性”和“落地性”时，才能称之为创新。以下是典型案例：

技术创新：新能源汽车的“刀片电池”——不仅是“新的电池形态”（新事物），更解决了传统锂电池“安全性低、能量密度不足”的痛点（价值性），通过“长电芯 + 蜂窝状结构”实现技术突破（突破性），并已大规模应用于量产车型（落地性）；

商业模式创新：共享单车——不仅是“新的出行工具组合”（新事物），更解决了“短途出行最后一公里”的需求（价值性），通过“分时租赁+物联网定位”重构了出行商业模式（突破性），并在全国范围内推广落地（落地性）；

服务创新：外卖平台的“准时宝”服务——不仅是“新的服务承诺”（新事物），更解决了用户“担心外卖迟到”的焦虑（价值性），通过“时间赔付机制 + 订单调度优化”突破了

# 1.3 应用创新思维解决方案的过程

# 创造性思维解决问题的过程:

□ 准备阶段：发现问题→分析问题→搜集问题→形成课题。  
□ 酝酿阶段：明确创新目标→继续收集资料→从事试验或研究→尝试各种想法的可行性。（若问题复杂，可能要经历多次失败的探求；当阻力很大时，则中断思维，但潜意识仍在大脑深层活动，等待时机）  
□ 顿悟阶段：是突破阶段。创造主题在特定情境下得到特定的启发，被唤醒（该阶段的作用机制比较复杂，一般认为是与长期酝酿所积蓄的思维能量有关，这种能量会冲破思维定势和障碍，使思维获得开放性、求异性、非显而易见性）  
□ 验证阶段：创造性思维不仅注重形式上标新立异，内容上也要求精确可靠

# 学科技能大赛 -- 桥梁项目大赛

- 目标：如何能获奖方法：运用发散思维和收敛思维  
- 桥梁结构，种类越多越好，但也要切合实际情况（如何发散，查看书籍、参观比赛、奇思妙想、展会）  
- 通过力学计算选出负载能力相对强的结构，当然支架、横跨梁的做法手艺也很关键，也有不同种类的，需要选择。  
- 团队分工合作要合理，有一阶段需要在规定时间里现场制作，团队合作就很重要（主次分明，分工协作，特长擅长）  
- 对作品进行加载试验，虚拟仿真，选出制作时间短、重量轻、承载强的桥梁参加比赛（如何收敛，对准目标，选择最优）
