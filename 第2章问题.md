## 思考题

### 问题：关于学术想象力培养方法的思考

**原题：** 关于前面所提到的学术想象力培养方法，你认可吗，对你来说影响最深刻是那些，你有更好的方法吗，若有，请进行说明。

**解题思路：**
本题要求结合课程内容，对想象力培养的四个核心方向进行反思：
1. 主动打破知识边界，为想象力积累"素材库"
2. 深度参与实践探索，让想象力在"试错"中落地
3. 刻意训练思维方式，为想象力搭建"方法论框架"
4. 营造"个性化"思维环境，让想象力"自由生长"

**参考答案：**

**一、对课程方法的认可与分析**

我认可课程提出的想象力培养方法，原因如下：

1. **主动打破知识边界**——这是想象力的"原料"来源
   - 跨学科知识涉猎能够提供不同视角
   - 案例佐证：钢筋混凝土的发明源于园艺家莫尼哀对植物根系的联想
   - 爱因斯坦说："想象力比知识更重要"，但想象力需要知识作为基础

2. **深度参与实践探索**——这是想象力的"检验场"
   - 实践能够验证想象的可行性
   - 案例佐证：霍尔通过实验验证了对麦克斯韦《电磁学》的质疑，发现了"霍尔效应"

3. **刻意训练思维方式**——这是想象力的"工具箱"
   - 发散思维、批判性思维、系统思维都是可训练的
   - 案例佐证：课程中的列举法（砖块新用途）就是发散思维的具体训练方法

4. **营造个性化思维环境**——这是想象力的"土壤"
   - 包容失败、预留"无目的时间"、选择支持创新的社交圈
   - 这与科学史上很多重大发现都是在放松状态下产生的规律相符

**二、影响最深刻的方法**

对我而言，**"批判性思维训练"和"主动提问"**影响最深刻：
- 课程中蒋有兴发现人类染色体是46个而非48个的案例说明：敢于质疑书本和权威是创新的前提
- "颜宁之问"启示我们：没有问题意识，就没有创新的起点

**三、补充方法建议**

1. **建立"问题银行"**
   - 随时记录日常学习和生活中遇到的问题、困惑、反常现象
   - 定期回顾，寻找问题之间的联系

2. **"思维实验"训练法**
   - 像爱因斯坦"追光实验"那样，在头脑中进行假设性实验
   - 不受实际条件限制，充分发挥想象

3. **"逆向教学"法**
   - 尝试向他人讲解自己学到的知识
   - 在讲解过程中发现知识漏洞，激发新问题

4. **"异领域案例迁移"法**
   - 主动学习其他领域的经典案例和解决方案
   - 思考如何将其迁移到自己的研究领域

## 课后作业

### 作业1：针对相关联想，谈谈乔布斯故事对你的启发

**解题思路：**
相关联想是由某个事物的出现而联想到与此有关的另外一些事物，主要体现于因果关系的运作应用。乔布斯的创新之路正是相关联想的典范。

**参考答案：**

**一、乔布斯的相关联想实践**

乔布斯曾说："创新就是把各种事物整合到一起。"他的创新方法正是相关联想的绝佳体现：

1. **书法课与Mac字体**
   - 乔布斯大学辍学后旁听了书法课
   - 多年后，他将书法美学融入Mac电脑，创造了漂亮的字体系统
   - 这是典型的相关联想：艺术美学→数字字体→用户体验提升

2. **音乐+电话+互联网→iPhone**
   - 乔布斯将iPod的音乐功能、手机的通讯功能、互联网的信息功能整合
   - 创造了改变世界的智能手机
   - 这是相关联想的"连锁反应"：单一功能→功能关联→整合创新

3. **皮克斯动画经验→苹果零售店**
   - 乔布斯从皮克斯学到"讲故事"的重要性
   - 将"故事体验"应用到苹果零售店的设计中
   - 相关联想：动画叙事→产品展示→沉浸式购物体验

**二、启发与应用**

1. **广泛积累是基础**
   - 乔布斯涉猎书法、音乐、禅学、商业等多个领域
   - 启发：要主动打破知识边界，积累联想的"素材库"

2. **敢于跨界是关键**
   - 他不把不同领域的知识割裂开来，而是寻找连接点
   - 启发：培养"万物互联"的思维习惯

3. **用户需求是导向**
   - 所有联想都指向一个核心：为用户创造更好的体验
   - 启发：相关联想要有明确的目标导向

4. **从"点"到"网"的思维升级**
   - 如课程所述："将各种因素相互连接起来，创意就由此而来"
   - 启发：建立系统思维，在复杂问题中找到关键节点

**三、我的相关联想训练计划**

1. 每周阅读一个专业外领域的内容
2. 建立"概念连接笔记"，记录不同领域概念的相似性
3. 定期进行"强制联想"训练：随机选取两个不相关的事物，寻找联系

### 作业2：描述未来手机

**原题：** 充分发挥想象，从功能、样式、材料等方面描述你认为的未来手机。

**解题思路：**
运用课程中介绍的想象方法：
- 相似联想：从自然界和其他领域获取灵感
- 相反联想：突破现有手机的局限
- 相关联想：思考技术发展的因果链条

**参考答案：**

**一、功能方面**

1. **全息投影交互**
   - 无需屏幕，手机可投射3D全息影像
   - 支持隔空手势操作，实现真正的"空中触控"
   - 联想来源：科幻电影+AR技术发展趋势

2. **神经接口连接**
   - 可与人体神经系统建立安全连接
   - 实现"意念操控"——想即所得
   - 联想来源：脑机接口技术（如Neuralink）的发展

3. **AI数字分身**
   - 手机内置个人AI助手，可学习并模拟用户的思维方式
   - 可代替用户处理日常事务、参加部分会议
   - 联想来源：大语言模型+个性化AI发展

4. **万物互联中枢**
   - 作为个人所有智能设备的统一控制中心
   - 自动协调家居、交通、健康等各系统
   - 联想来源：物联网技术整合

5. **实时翻译与情感感知**
   - 支持任何语言的实时双向翻译
   - 能感知对方情绪并给出沟通建议
   - 联想来源：NLP技术+情感计算

**二、样式方面**

1. **形态可变**
   - 可折叠、卷曲、拉伸，根据需要变换形态
   - 手表模式、手机模式、平板模式自由切换
   - 联想来源：柔性屏技术+仿生设计（变色龙）

2. **可穿戴融合**
   - 可化为手镯、戒指、眼镜等形态
   - 与服装面料融合，成为"智能服饰"
   - 联想来源：穿戴设备发展趋势

3. **透明/隐形模式**
   - 采用透明材料，不用时可"消失"于环境中
   - 联想来源：隐身材料研究+极简设计理念

4. **个性化外观**
   - 外壳颜色、图案可通过电子墨水实时改变
   - 支持用户情绪感应自动变色
   - 联想来源：电子纸技术+情感化设计

**三、材料方面**

1. **自修复材料**
   - 采用仿生自愈合材料，小划痕可自动修复
   - 联想来源：生物皮肤的再生能力

2. **可降解环保材料**
   - 主体采用可生物降解的环保材料
   - 电子元件可完全回收再利用
   - 联想来源：环保理念+绿色制造

3. **超级电池/无线取能**
   - 石墨烯电池：充电5分钟，使用一周
   - 或采用空间无线能量传输，永不需充电
   - 联想来源：电池技术突破+无线充电发展

4. **抗菌材料**
   - 表面采用纳米抗菌涂层
   - 自动杀灭99.9%的细菌和病毒
   - 联想来源：后疫情时代的健康需求

**四、商业模式创新**

1. **手机即服务（Phone as a Service）**
   - 不再购买手机，而是订阅服务
   - 硬件自动升级更新

2. **模块化设计**
   - 用户可根据需要更换/升级特定模块（摄像头、电池等）
   - 延长手机使用寿命，减少电子垃圾

### 练习1：公安局长下棋的逻辑题

**原题：** 一位公安局长在茶馆里和一位老头下棋。正下到难解难分之际，跑来一位小孩，小孩着急地对公安局长说："你爸爸和我爸爸吵起来了。"老头问："这孩子是你的什么人？"公安局长回答道："我儿子。"请问这两个吵架的人与公安局长是什么关系？

**解题思路：**
这道题考查的是克服"思维定式"——大多数人会习惯性地认为公安局长是男性。
- 关键突破点：公安局长可以是女性
- 分析"你爸爸和我爸爸"的关系

**答案：**
公安局长是**女性**。
- 小孩是公安局长的儿子
- "你爸爸"指的是公安局长的丈夫（小孩的爸爸）
- "我爸爸"指的是公安局长的父亲（小孩的外公）

因此，两个吵架的人是：
- 一个是公安局长的**丈夫**
- 一个是公安局长的**父亲**

### 练习2：酒瓶软木塞问题

**原题：** 宴会上，主人拿出一瓶酒，告诉大家，酒瓶用软木塞塞住，没有起塞工具又不准损坏瓶子，能否使大家喝到瓶子里的酒呢？

**解题思路：**
打破思维定式——塞子不一定要"起出来"，可以换个方向思考。

**答案：**
**将塞子塞入瓶中**，酒便能倒出来。

**分析：**
- 常规思维：塞子只能往外拔
- 创新思维：塞子也可以往里推
- 这体现了课程中"排除自设，打破僵化结构"的思维方法

### 练习3：汽车过桥洞问题

**原题：** 一辆满载货物的汽车要通过一座铁桥，通过时发现货物高于桥洞一厘米，现在问你，在不准卸货重装的情况下，能让车通过吗？

**解题思路：**
打破"只能在货物上想办法"的思维定式，可以在车辆本身想办法。

**答案：**
**给轮胎放掉一点儿气**，降低车身高度，车便能通过。

**分析：**
- 常规思维：在货物高度上想办法
- 创新思维：改变车的整体高度
- 只需放掉少量气体，车身下降1厘米以上即可

### 练习4：六个杯子排列问题

**原题：** 有六个普通的杯子在桌子上排成一排。头三个里有水，后三个是空的。要求你只移动或变换一个杯子，使杯子的排列由"水、水、水、空、空、空"变成"水、空、水、空、水、空"的排列，怎么办？

**解题思路：**
- 常规思维：如何移动杯子的位置
- 创新思维：可以移动杯子里的水

**答案：**
**将第二个杯子的水倒入第五个杯子**，然后将第二个杯子放回原位。

**分析：**
- 原排列：水(1)、水(2)、水(3)、空(4)、空(5)、空(6)
- 操作：拿起第2个杯子，把水倒入第5个杯子，放回第2个杯子
- 结果：水(1)、空(2)、水(3)、空(4)、水(5)、空(6)
- 关键突破：把思维重心从"移动杯子"转到"移动水"上

这是"相关联想"的应用——将"搬家"与"借还书流程"联系起来，化繁为简。

## 测试题

### 测试1：鲁钦斯量水实验

**原题：** 给定三个容器A、B、C，求量取指定水量D。找出简便解法。

| 问题 | A | B | C | 求D | 简便解法 |
|-----|---|---|---|-----|---------|
| 1 | 21 | 127 | 3 | 100 | |
| 2 | 14 | 163 | 25 | 99 | |
| 3 | 18 | 43 | 10 | 5 | |
| 4 | 9 | 42 | 6 | 21 | |
| 5 | 20 | 59 | 4 | 31 | |
| 6 | 23 | 49 | 3 | 20 | |
| 7 | 15 | 39 | 3 | 18 | |
| 8 | 18 | 48 | 4 | 22 | |
| 9 | 14 | 36 | 8 | 6 | A-C |

**解题思路：**
这个实验展示了思维定式的形成过程。前几题都可以用公式 **D = B - A - 2C** 解决，但后面的题目有更简便的方法。

**答案：**

| 问题 | 通用解法 B-A-2C | 简便解法 |
|-----|----------------|---------|
| 1 | 127-21-6=100 ✓ | B-A-2C |
| 2 | 163-14-50=99 ✓ | B-A-2C |
| 3 | 43-18-20=5 ✓ | B-A-2C |
| 4 | 42-9-12=21 ✓ | B-A-2C |
| 5 | 59-20-8=31 ✓ | B-A-2C |
| 6 | 49-23-6=20 ✓ | **A-C=23-3=20** |
| 7 | 39-15-6=18 ✓ | **A+C=15+3=18** |
| 8 | 48-18-8=22 ✓ | **A+C=18+4=22** |
| 9 | 无法使用 | **A-C=14-8=6** |

**分析：**
- 问题1-5形成了"B-A-2C"的思维定式
- 问题6-8用定式解法也能得到答案，但有更简便的方法
- 问题9必须打破定式才能解决
- 这说明思维定式的二重性：有利于快速解决常规问题，但会阻碍发现更优解法

### 测试2：五边形地砖铺设

**原题：** 如何使用五边形地砖铺贴？

**解题思路：**
打破思维定式——五边形不一定是"正五边形"。

**答案：**
正五边形（内角108°）无法单独密铺平面（360°不能被108°整除），但：

1. **不规则五边形可以密铺**
   - 目前已发现15种可以密铺平面的凸五边形
   - 2015年才发现第15种，由数学家通过计算机搜索发现

2. **常见解决方案：**
   - 使用特殊设计的不规则五边形
   - 多种五边形组合铺设
   - 五边形与其他形状（如三角形、四边形）组合

**启示：**
大多数人受到"五边形=正五边形"的惯性思维限制。这道题提醒我们：要质疑隐含的假设条件。

### 测试3：四个1组成最大数

**原题：**
- 由两个1所能排列成的最大的数是多少？→ 11
- 由三个1所能排列成的最大的数是多少？→ 111
- 由四个1所能排列成的最大的数是多少？

**解题思路：**
前两题建立思维定式，第三题需要打破。

**答案：**
**11^11 = 285,311,670,611**（约2853亿）

而不是1111。

**分析：**
- 常规思维：四个1并排 → 1111
- 创新思维：使用指数 → 11的11次方
- 这是"排除自设"的典型案例——没人规定数字只能并排写

### 测试4：九点连线问题

**原题：** 在一张纸上画出等距离排列成三行三列的九个点，用四条直线一笔将所有的点连接起来，同一点不可重复经过。

**解题思路：**
打破"线段必须在九个点围成的方框内"的隐含假设。

**答案：**
```
    1 ● ─ ─ ─ ● ─ ─ ─ ● ─ ─ →
              │
    2 ● ─ ─ ─ ● ─ ─ ─ ●
              │       ↗
    3 ● ─ ─ ─ ● ─ ─ ─ ●
      ↖               │
        ← ─ ─ ─ ─ ─ ─ ↲
```

**图解步骤：**
1. 从左上角开始，斜向右下穿过三点，延伸到方框外
2. 向左画线，穿过底部三点，延伸到方框外
3. 斜向右上穿过三点，延伸到方框外
4. 向下画线，穿过右侧三点

**启示：**
- 我们自设了"线不能超出九点范围"的限制
- 解决问题需要"跳出框框"思考（Think outside the box）
- 这是"排除自设，打破僵化结构"的经典案例

### 测试5：六根火柴组成四个等边三角形

**原题：** 用六根火柴试试能否摆出彼此相连的四个等边三角形？

**解题思路：**
打破"在平面上摆放"的思维定式。

**答案：**
**搭建成三维的正四面体（金字塔形状）**

```
        ●
       /|\
      / | \
     /  |  \
    ●───┼───●
     \  |  /
      \ | /
       \|/
        ●
```

**分析：**
- 在平面上，六根火柴最多只能组成2个等边三角形
- 突破到三维空间，六根火柴恰好能搭成正四面体
- 正四面体有4个面，每个面都是等边三角形
- 这是"打破维度限制"的典型案例

### 测试6：红枣问题

**原题：** 将20个红枣放入摆在桌上的三个同样大小的碗里，要求每个碗里的红枣都是单数。问：红枣该怎么放？

**解题思路：**
- 数学分析：三个单数相加必定是单数（单+单+单=单），而20是偶数
- 常规条件下无解
- 需要打破"三个碗并列放置"的隐含假设

**答案：**
**把其中两个碗叠起来**

原题的实际解法：**让碗的嵌套改变"里面"的定义**
- 把碗B放进碗A里
- 碗B里放7个红枣（单数）——从碗B角度
- 碗A里（算上碗B里的）= 7 + 碗A自身放的
- 如果碗A自身放6个，则碗A里有13个（单数）
- 碗C放7个（单数）
- 总计：6+7+7=20 ✓

**最终答案：**
- 将碗B放入碗A中
- 碗A自身（碗B外部）放6个
- 碗B里放7个
- 碗C放7个
- 从每个碗角度：
  - 碗A里有 6+7=13个（单数）✓
  - 碗B里有 7个（单数）✓
  - 碗C里有 7个（单数）✓
- 总计：6+7+7=20 ✓

**启示：** 题目没有规定碗必须并排放置，打破这个隐含假设即可解决。

## 案例分析题

### 案例1：双绳问题（梅尔实验）

**问题描述：** 房间天花板上垂下两根绳子，要把它们系在一起，但两绳相距较远，用手抓住一根绳子后无法够到另一根。房间里有钳子等工具。

**解题思路：**
打破"钳子只能用来夹东西"的功能固着。

**答案：**
把钳子系在一根绳子末端作为摆锤，使绳子摆动起来。然后抓住另一根绳子走到中间，在摆动的绳子荡到最近点时接住它，将两根绳子系在一起。

**分析：**
- 功能固着让我们认为钳子只有"夹"的功能
- 创造性思维发现钳子还有"重量"这个属性，可以作为摆锤

---

### 案例2：蜡烛问题（登克尔实验）

**问题描述：** 桌上有一盒图钉、几根火柴和蜡烛，要把蜡烛固定在门上。

**解题思路：**
打破"图钉盒只是容器"的功能固着。

**答案：**
把图钉倒出来，用图钉把空盒子钉在门上作为"支架"，再把蜡烛粘在盒子里或放在盒子上。

**分析：**
- 当图钉放在盒子里时，人们只把盒子看作"容器"
- 把图钉倒出后，更容易发现盒子可以作为"托盘/支架"

---

### 案例3：聋哑人买钉子

**问题描述：** 一位聋哑人去商店买钉子，因无法用语言表达，他用一只手做握锤状，另一只手做握钉状，并用握锤的手做敲击动作，售货员给了他锤子。他摇头，指了指"钉子"方向的手指，售货员明白后给了他钉子。随后一位盲人来买剪刀，他会怎样表达？

**常见错误答案：** 用手比划剪刀的动作。

**正确答案：** **盲人会直接开口说"我要买剪刀"。**

**分析：**
- 聋哑人的故事建立了"用动作表达需求"的思维定式
- 盲人虽然看不见，但并不影响说话
- 这是思维定式的典型陷阱

---

### 案例4：竹禅画九尺观音

**问题描述：** 慈禧太后要在五尺宣纸上画一幅九尺高的观世音菩萨站立像，如何完成？

**常规思维困境：** 五尺纸怎么能画九尺高的像？

**答案：**
竹禅和尚画的观音是**弯腰拾柳枝**的姿态。观音弯着腰，如果直起腰来正好九尺高。

**分析：**
- 打破"画人必须画站直状态"的思维定式
- 通过改变姿态，在有限空间表达更大的意象
- 这是创新思维"排除自设条件"的典型案例

## 总结：第2课核心知识点

### 一、创新应具有的四大基本素质

1. **品德素质**——创新的道德底线
2. **创新意识**——创新的内在动力
3. **思维定式的克服**——创新的障碍突破
4. **想象力**——创新的核心能力

### 二、思维定式的五种类型

| 类型 | 定义 | 克服方法 |
|-----|-----|---------|
| 习惯性思维 | 用旧方法处理新问题 | 认清经验局限，勇于尝试新方法 |
| 从众思维 | 盲目跟随众人 | 保持独立思考，敢于做出相反决定 |
| 权威思维 | 对权威的盲从 | 质疑权威，以实证为准 |
| 自我中心思维 | 只从自己角度出发 | 换位思考，考虑多方视角 |
| 书本型思维 | 不顾实际盲从书本 | 实践检验，敢于质疑书本 |

### 三、克服思维定式的四种方法

1. **认清经验的局限**——未知永远大于已知
2. **慎对从众**——从集体思考中跳出来
3. **突破功能固着**——发现旧事物的新用途
4. **排除自设**——打破僵化的结构

### 四、想象力培养的三种联想方法

1. **相似联想**：从相似事物获取启发（如仿生设计）
2. **相反联想**：对已知事物进行相反思考（如司马光砸缸）
3. **相关联想**：从因果关系延伸思考（如黑色玩具猫的故事）

### 五、研究生想象力培养的四个核心方向

1. 主动打破知识边界，积累"素材库"
2. 深度参与实践探索，在"试错"中落地
3. 刻意训练思维方式，搭建"方法论框架"
4. 营造"个性化"环境，让想象力"自由生长"
