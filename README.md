# 创造性思维课程资料

> 打破常规、蓄智之能、激发创意

本仓库收录《创造性思维/创新思维》课程的完整学习资料，包括讲义笔记、章节习题及 TRIZ 创新方法专题。内容以 Markdown 格式编写，便于检索、学习与二次整理。

---

## 📚 课程简介

本课程旨在培养创新型人才，系统讲授创新思维的特征、规律与方法，帮助学习者：

- 树立正确的创新观，理解创新的本质与价值
- 掌握发散思维、收敛思维、逻辑思维等多种思维方式
- 学习头脑风暴法、模块化、延伸法、移植法等创新技法
- 深入理解 TRIZ（发明问题解决理论）的核心工具与方法

**课程目标**：培养学生的创新思维能力和悟性，训练发明创造技能，引导大胆创新。

## 🎯 学习指南

### 推荐学习路径

```
基础阶段（第1-4讲）
    ↓
掌握创新基本概念、素质要求、基础思维方法
    ↓
技法阶段（第5-6讲）
    ↓
学习各类创新技法及其实际应用
    ↓
TRIZ专题（第7-9讲 + TRIZ 40个发明原理详解）
    ↓
深入理解系统化创新方法论
```

### 学习建议

1. **循序渐进**：按课程顺序阅读讲义，每讲完成后做对应章节习题
2. **理论结合实践**：尝试将所学方法应用于实际问题
3. **TRIZ 重点突破**：第 7-9 讲与 `TRIZ 40个发明原理详解.md` 建议搭配学习
4. **案例分析**：讲义中包含大量实例，注意理解其背后的思维逻辑

### 核心知识点速览

- **创新能力公式**：创新能力 = K ×（创造性人格 + 创造性思维 + 创新方法）× 知识量²
- **TRIZ 理想度**：理想度 = Σ有用功能 /（Σ有害作用 + 成本）
- **技术系统进化 S 曲线**：婴儿期 → 成长期 → 成熟期 → 衰退期
- **八大进化法则**：完备性、能量传递、协调性、提高理想度、动态性、子系统不均衡进化、向微观级进化、向超系统跃迁
