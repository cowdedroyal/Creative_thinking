# 创造性思维课程资料

本仓库存放《创造性思维/创新思维》课程的讲义笔记、章节问题与参考解答，以及 TRIZ 相关整理。内容以 Markdown 形式编写，便于检索与二次整理。

## 内容结构

- 课程讲义：`第1课 创新思维及训练方法概论.md` 至 `第9讲 物理矛盾与技术矛盾解决原理.md`
- 章节问题与解答：`第1章问题.md` 至 `第9章问题.md`（第7、8章另有多版本整理）
- TRIZ 专题：`TRIZ 40个发明原理详解.md`
- 其他文章：`从研究生自身角度谈想象力的培养方法.md`

## 章节问题文件对照

- 第1、2、3、4、5、6、8、9章：`第1章问题.md` 等按章命名文件
- 第7章多版本：`123123.md`、`2222.md`、`256256.md`
- 第8章多版本：`111111.md`、`33333.md`、`6666.md`

## 使用方式

1. 按课程顺序阅读“第1课/第X讲”文件。
2. 对照“第X章问题.md”或对应多版本整理进行复习。
3. TRIZ 相关内容建议与第7–9讲搭配阅读。
