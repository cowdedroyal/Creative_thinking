# 5.1 头脑风暴法

头脑风暴法：(Brain Storming) 又称为

“狂想法”、“智力激励法”、

“诸葛亮会”、“畅谈会法”、

“BS 会议法”等

![](https://cdn-mineru.openxlab.org.cn/result/2025-12-13/ff9173be-9c18-470e-8c39-9a6cf284bae1/18accc03868ba7492002ddfd2ba90b03c00ed0a11a0aa0db671e329542e88079.jpg)

头脑风暴法是一种有助于集思广益的集体思考的方法。

这种方法已为一些国家所广泛应用。奥斯本在《创造性想象》一书中指出，这种方法能够在社会、经济、管理、教育、新闻、科技、军事、生活等很多方面提供有效的服务。

- 人们要创造和创新，首先要能想到较多较好的设想、方案、“点子”，而为了产生这些新设想、新方案，通过一定的会议形式，营造能够相互启发、引起联想、发生“共振”的条件与机会，便有助于开发人们的智慧和创造力。  
- 头脑风暴法就是通过这种会议形式以激励人们创造力的一种方法。它能在较短的时间里发挥集体的创造力从而获得较多的创造设想，激发组内其他成员的联想能力，当人们卷入“头脑风暴”的洪流之后，就会引起一系列的设想，这就象放一串鞭炮一样，点燃一个爆竹，就会引爆一连串的爆竹。

# 头脑风暴法的具体步骤

(1) 召集一种特殊会议, 与会人数 5—10 人之间。人数多了不便充分发表意见。如要吸收更多人的意见, 可召开几个会。  
(2) 会议有一名主持者, 有  $1 - 2$  名记录员, 后者最好不是正式参加会议的人员。  
(3) 会议时间以一小时为限，一般在半小时至一小时之间。时间长了脑子容易疲劳。开会时思想要高度集中。  
(4) 会议地点应选择安静而不受外界干扰的场所。切断电话, 谢绝会客。  
(5) 事先通知。

# 头脑风暴法的四项规则

(1) 自由奔放。提倡任意、自由思考, 自由想象, 想法越新奇越好。有时看起来是很荒唐的设想却可能很有价值。  
②延迟评价。严禁批判，对别人提出的想法不能批评，不得阻拦，即使是幼稚的、错误的、荒诞无稽的，也不得批判。也就是说，要排除评论性的判断，对设想的评论要放在以后进行。这叫延迟评价原则。③追求数量。要求提出一定数量的设想，提出的设想越多，就越可能获得更多的有价值的设计和方案。  
(4) 善于利用别人的想法开拓自己的思路。

# 头脑风暴法四项规则的普遍意义

(1) 自由奔放：体现了“创造”需要科学的自由精神和质疑精神。  
(2) 延迟评价: 为 “创造” 提供一个有利的政治和文化的环境。  
(3) 追求数量: 体现了 “创造” 的低概率本质。  
(4) 善于利用别人的想法开拓自己的思路: 体现了 “创造” 的继承性和社会性。

头脑风暴练习：美国北方格外严寒，大雪纷飞，电线上积满冰雪，大跨度的电线常被积雪压断，严重影响通信。

按照规则，大家七嘴八舌议论开来。有人提出设计一种专用的电线清雪机，有人想到用电热来化解冰雪，也有人建议用震荡技术来清除积雪，还有人提出能否带上几把大扫把，乘直升飞机去扫电线上的积雪。对于这种“坐飞机扫雪”的设想，大家尽管觉得滑稽可笑，但在会上也无人提出批评。相反，有一位工程师在百思不得其解时，听到用飞机扫雪的想法后，大脑突然受到冲击，一种简单可行且高效率的清扫方法冒了出来。他想，每当大雪过后，出动直升机沿着积雪严重的电线飞行，依靠告诉旋转的螺旋桨即可将电线上的积雪扇落。他马上提出“用直升机扇雪”的新设想，顿时又引起其他与会者的联想，有关用直升机除雪的主意一下子又多了七八条。不到一个小时，与会的10名技术人员共提出90多条新设想。

会后，公司组织专家对设想进行分类论证，专家们认为设计专用的清雪机，采用电热或电池震荡等方法清理电线上的积雪，在技术上虽然可行，但研制费用大，周期长，一时难以见效。那种因“坐直升机扫雪”激发的几种设想，倒是一种大胆的新方案，如果可行，将是一种既简单又高效的好办法。经过现场试验，发现用直升机扇雪真能奏效，一个久悬未决的难题，终于在头脑风暴中得到了巧妙解决。

一些国家的创造学家在奥斯本头脑风暴法的基础上,又发展出了许多类似的方法:

1. 默写式头脑风暴法  
2. 卡片式头脑风暴法（又分CBS法和NBS法两种）  
3. 函询集智法

# 函询集智法

基本原理：借助信息反馈，反复征求专家书面意见来获得新的创意。

基本作法：就某一课题选择若干名专家作为函询调查对象，以调查表形式将问题及要求寄给专家，限期索取书画回答。收到全部复函后，将所得设想或建议加以概括，整理成一份综合表。然后，将此表连同设想函询表再次寄给各位专家，使其在别人设想的激励启发下提出新的设想或对已有设想予以补充或修改。视情况需要，经过数轮函询，就可得到许多有价值的新设想。

# 函询集智法的运用程序

1）专家选聘。此项工作是运用这一技法成败的关键。专家类型要精博结合，一般可从部门或系统内外挑选。所选专家必须对函询活动和主题有兴趣，乐意承担任务并能坚持始终。专家人数要根据欲解决问题的性质、规模和要求而定，从几人到几十人不等。  
2) 函询调查表的编制。调查表示此法运用中的主要信息载体和通道，其质量对结果影响很大。编制调查表要以方便专家为原则。表上所列为题尽可能分门别类，要求明确；表要简化，便于专家理解和填写；要确定专家应答方式；不要诱导专家按自己的设想作答。  
3) 函询调查的组织。此阶段是实施技法的实质阶段，函询过程中要创造条件让专家能够自由思考，独立判断。应注意搞好信息反馈工作。为保证函询匿名的特点，只能将整理好的综合设想反馈给专家，同时允许专家在后一轮回答中修改前一轮回答的意见或设想。  
4) 设想的加工整理。对专家提出的设想在每一轮进行之后都要进行加工整理，最后一轮结果的加工整理更需要认真进行。常需统计分类，归纳概括，发展完善，最后评选出可供采用的新设想。

# 例：家用电器新产品开发课题的策划！

1）根据开发研究要求，选聘国内外30位专家担任函询对象；  
2）制定“家用电器新产品开发课题函询表”。表中列出家用电器基本类别，划分出“重点考虑”、“一般考虑”、“暂不考虑”等级别，并对其做出相应说明。  
3）进行轮间反馈。第一轮函询表，收到150个家电产品新设想，经归纳整理成80个新设想再次进行函询。第二轮函询表收回后，专家们设想集中到重点考虑的课题45个。经第三轮函询后，专家把急需开发的重点家用电器产品集中到6类20种。  
4）最后根据专家反馈的设想，制定出家用电器产品重点研究开发计划。

# 5.2 模块化

模块化：解决一个复杂问题时，自上而下逐层把系统划分成若干模块的过程。

$\succ$  是一种处理问题方法的创新。  
>生活中到处都有模块化的影子。

模块化创新方法—利用模块化方法重新搭建事物，通过模块之间的适当的搭配，创造出具有不同特征的、满足不同需求的多种多样的产品。

又叫“另类的组合法”

如：个人电脑 = CPU+ 主板 + 内存 + 硬盘 + 显示器 +……（最适合自己）

# 实例 4:Framework Laptop 笔记本电

# 脑

Framework Laptop 是美国 Framework 公司推出的主打全模块化设计的消费级笔记本电脑产品，其本质是通过硬件架构重构，打破传统笔记本“一体化不可升级、维修难、淘汰快”的行业痛点，打造“可长期使用、可自主掌控、可持续环保”的新型电子设备范式，具体可从以下三大核心维度理解：

# 一、产品定位：模块化创新的“标杆载体”

它并非普通笔记本电脑，而是以“模块化”为核心卖点的创新产品——区别于传统笔记本将CPU、主板、屏幕等硬件焊接或集成一体的设计，Framework Laptop从底层架构上就将硬件拆分为独立功能模块，用户可像“拼积木”一样自主更换、升级核心部件，本质是通过“硬件模块化”解决电子设备“短期淘汰”的行业难题，成为消费电子领域模块化创新的典型实践案例。

# 二、设计内核：三大核心特征的集中体现

其设计围绕“可升级、可维修、可持续”展开，每个特征都对应具体的模块化设计：

1. 可升级的硬件架构：将传统一体化主板拆分为 CPU、GPU、PCH（平台控制器中枢）三个独立模块，支持单独插拔升级（如从 i5 处理器模块更换为 i7 模块），同时屏幕、内存、硬盘、电池等高频迭代 / 易损部件也均为独立可替换模块，避免因单一硬件老化导致整机淘汰；  
2. 低门槛的维修体验：采用标准化接口（如统一的模块连接端子）和通用工具（普通十字螺丝刀），配合可视化拆解指南，非专业用户也能在1小时内完成模块更换（如自行更换故障电池），打破传统笔记本“维修依赖官方、成本高、周期长”的壁垒；  
3. 可持续的环保逻辑：通过“模块复用 + 单独升级”延长整机寿命（从传统笔记本的3-5年延长至7-10年），减少因整机淘汰产生的电子垃圾；同时采用无焊接结构、可回收材料，经测算其全生命周期碳足迹较传统笔记本降低  $58\%$  ，契合全球“减碳环保”趋势。

# 三、行业意义：推动电子设备“范式变革”

Framework Laptop 的更深层含义，是为消费电子行业提供了“模块化创新的可落地模板”——它证明了模块化设计在消费级产品中的可行性：既满足用户对硬件性能升级、个性化定制的需求（如办公场景选低功耗CPU，设计场景换高性能GPU），又能通过“开源协作”（开放部分设计图纸、接口规范）激活用户与开发者共创（如第三方开发扩展模块），甚至带动其他电子设备（如智能手机、平板电脑）借鉴其模块化逻辑，推动整个行业向“可持续、可循环”的方向转型。

简言之，Framework Laptop 不止是一款笔记本，更是“以用户为中心、以环保为导向”的模块化设计理念的“实体化呈现”，其核心价值在于重新定义了电子设备的“使用逻辑”——从“短期更换”变为“长期陪伴”，从“被动接受”变为“主动掌控”。

- 模块：一组具有同一功能和接合要素（指联接部位的形状、尺寸、联接件间的配合或啮合等），但性能、规格或结构不同却能互换的单元。  
- 模块化创新设计：在对产品进行市场预测、功能分析的基础上，划分并设计出一系列通用的功能模块；根据用户的要求，对这些模块进行选择和组合，就可以构成不同功能的产品。这种设计方法称为模块化设计。

# 模块化创新设计三大特点

# - 相对独立性

可以对模块单独进行设计、制造，调试、修改和存储，这便于由不同的专业化企业分别进行生产

# - 通用性

模块接口部位的结构、尺寸和参数标准化，容易实现模块间的互换，从而使模块满足更大数量的不同产品的需要

# - 互换性

有利于实现产品间模块的通用，实现跨系列产品间模块的通用

# 典型应用案例

- 微信：核心功能“即时通讯”→功能延伸（微信支付、小程序、视频号）→场景延伸（工作沟通、生活缴费、内容消费）→从社交工具升级为生态平台。  
- AirPods：核心功能“无线听歌”  $\rightarrow$  功能延伸（主动降噪、空间音频、查找设备）  $\rightarrow$  场景延伸（办公通话、运动听歌、游戏音效）  $\rightarrow$  成为全场景音频设备。  
- 小米充电宝：核心功能“移动充电”  $\rightarrow$  功能延伸（自带 Type-C 线、双向快充）  $\rightarrow$  形态延伸（超薄便携款、户外大容量款）  $\rightarrow$  覆盖日常通勤、户外露营等多场景。  
- 宜家马尔姆系列：核心功能“收纳储物”  $\rightarrow$  形态延伸（单抽屉柜  $\rightarrow$  双抽屉柜  $\rightarrow$  组合式书柜）  $\rightarrow$  场景延伸（卧室收纳  $\rightarrow$  客厅储物  $\rightarrow$  书房陈列）  $\rightarrow$  适配不同户型需求。

定义：把原有事物的功能、结构、状态、用途等增加或扩大，以获得更多、更强、更好的效果。

两种形式：延伸和扩展

延伸 -- 对原有事物进行再创造使之更为完美；

扩展 -- 利用现有的技术，解决新问题。

- 锚定核心根基：不脱离原有产品的核心功能、技术或用户群体，降低研发与市场接受风险。  
- 聚焦需求缺口：围绕用户未被满足的“关联需求”延伸，而非无中生有（例：手机拍照→专业摄影模块）。  
- 单一维度突破：优先选择 1 个核心延伸方向（功能 / 场景 / 形态），避免多维度分散资源。

# 实施步骤（4步落地）：

1. 拆解核心要素：明确现有产品的核心功能（如充电宝“充电”）、核心技术（如无线连接）、核心用户（如年轻上班族）。  
2. 挖掘延伸缺口：通过用户反馈、场景分析，找到“核心需求之外的关联痛点”（如充电宝用户需“自带线、应急照明”）。  
3. 选择延伸维度：聚焦1-2个高价值维度发力，常见维度包括：

功能延伸：核心功能 + 附加功能（如空调→空调 + 空气净化 + 除湿）。

场景延伸：原有场景  $\rightarrow$  新场景（如跑步机  $\rightarrow$  家用跑步机 + 折叠便携款 + 办公室迷你款）。

形态延伸：原有形态  $\rightarrow$  适配新需求的形态（如普通雨伞  $\rightarrow$  折叠伞  $\rightarrow$  反向伞  $\rightarrow$  晴雨两用伞）。

技术延伸：核心技术  $\rightarrow$  跨领域复用（如无人机导航技术  $\rightarrow$  车载自动驾驶辅助）。

4. 小步验证迭代：先推出简化版延伸产品，收集用户反馈后再优化（如先给充电宝加自带线，再升级双向快充）。

# 注意事项:

- 避免“过度延伸”：偏离核心优势的延伸会稀释品牌认知（如手机厂商盲目延伸做家电，导致资源分散）。  
- 拒绝 “无需求延伸”：延伸功能需匹配用户真实痛点，而非为了创新而创新（如给老年机加复杂游戏功能）。  
- 控制延伸成本：优先复用现有技术和供应链，降低研发与生产门槛（如充电宝延伸自带线，无需新增核心技术）。

延伸法应用比较简单。甚至不需要理解原有功能的原理、结构、如何实现等等。希望同学们深入理解这种方法的特征，多多练习，灵活运用。

# 瞬间压力差原理及应用

- 问题：甜椒的籽和把与果肉连在一起，形成一个完整的甜椒，但是我们想食用甜椒时，都会先将甜椒的把和籽去掉，这样吃起来口感更好。几个甜椒去籽、把很简单，人工处理一下就可以解决，但要一次性处理上万个甜椒要怎么办呢？

可以将甜椒放到一个密闭容器中，然后慢慢加大气压，随着气压加大，甜椒内部的气压也开始加大，然后将容器的阀门突然打开，甜椒的籽、把就会爆出来。

![](https://cdn-mineru.openxlab.org.cn/result/2025-12-13/ff9173be-9c18-470e-8c39-9a6cf284bae1/d96f9277c81a16116840d15e6793112583767d0702efd0b056bfb76591d4a8a5.jpg)

应用瞬间压力差原理就产生了200多种发明，典型的有迅速剥除松子壳的发明、使钻石沿内部原有的微裂纹分割的发明、船用发动机冷却水过滤器的快速清洁系统等

冷光源原理是在电场作用下产生电子碰撞激发荧光材料而产生的发光现象。

萤火虫发光特性启示, 从萤火虫的发光器中成功提取了荧光素和荧光素酶,通过分析了解其成分, 再通过化学合成方法成功制备出这些物质, 应用在发光设备上, 大大提升了发光效率。

如日光灯、霓虹灯、液晶显示屏、LED等，都是得益于萤火虫的启发而发明出的冷光源。当然，冷光源并不是一点热量也不会产生，只是发光效率提升了很高，比如日光灯就比白炽灯提升了  $35\%$  ，而LED则提升了  $75\%$  。

仿生移植法以自然界生物为灵感源，通过观察生物适应环境的生存策略（如形态优化、高效功能、节能机制等），提取其背后的科学原理，再将这些原理转化为产品设计、技术方案或流程优化的创新技法。

# 核心本质

- 不是 “复制生物外形”, 而是 “移植底层原理”: 比如模仿鸟的飞行, 核心是移植 “流体力学升力原理”, 而非单纯做 “翅膀形状”。  
- 跨领域的原理复用：将生物界经过亿万年进化的“最优解”，迁移到工业、科技、医疗等非生物领域，降低创新试错成本。

# 实施关键步骤

1. 观察生物特性：聚焦生物在生存、运动、捕食等场景中的核心优势（如鲨鱼的减阻、蜂巢的承重、蝙蝠的定位）。  
2. 提炼核心原理：拆解生物特性背后的科学逻辑（如蜂巢的正六边形结构  $\rightarrow$  “最小材料实现最大空间 + 最强承重”）。  
3. 技术转化落地：将原理与目标领域的技术、需求结合，转化为可实现的产品 / 方案（如正六边形原理  $\rightarrow$  蜂窝状建筑材料）。  
4. 验证优化：测试转化后的方案，结合实际场景调整（如仿生机器人的步态优化）。

# 5.5 联想创造发明法

联想是由一事物想到另一事物的心理过程。

创造性思维的本质在于发现原来以为没有联系的两个事物（或现象）之间的联系。人们要想发现这种联系，联想起着极为重要的作用。

《今古奇观》中有个苏小妹三难新郎的故事，说的是新郎人洞房前要经过“考试”，考试的内容是对对子，对不上来不许入洞房。第一对，第二对，都对上了，通过了，这时苏小妹又出了第三对的上联：“双手推开窗前月”，叫新郎秦观对下联。秦观想了良久，依然百思不得其对，苏东坡看得心里着急，有心助他一臂之力，便向池里投了一块石头，水面上泛起了波纹就象划破了水中的天空，秦观见此情景，顿时大悟，立刻对出了一句：“一石击破水中天”。在这里，联想帮了秦观的大忙。
