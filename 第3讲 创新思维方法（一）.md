项目背景：某公司计划开发一款面向年轻群体的智能手表，目标是提供健康监测、运动追踪和音乐控制等功能。

# 需求分析阶段---发散思维：

在需求分析阶段，团队首先运用因果发散分析用户需求的深层原因。为什么年轻人需要智能手表？原因可能包括：健康意识提升、社交需求、个性化表达、科技追求等。基于这些分析，团队进一步运用关系发散，探索智能手表与用户生活的各种联系：与手机的连接、与运动的结合、与时尚的融合、与健康管理的关系等。

# 设计阶段---发散思维：

在设计阶段，团队运用了多种发散思维形式：

材料发散：在选择手表材料时，考虑了金属、塑料、橡胶和新型复合材料等多种选项功能发散：从健康监测功能出发，设想了心率监测、睡眠监测、压力监测、步数统计等多种功能

形态发散：在外观设计上，考虑了圆形、方形、长方形等形状，以及各种颜色搭配组合发散：思考手表可以与哪些功能组合，如与 GPS 定位、支付功能、语音助手等组合

# 开发阶段---发散思维：

在技术开发阶段，团队运用方法发散，探索实现各项功能的不同技术路径。例如，对于心率监测功能，考虑了光电传感器、心电图、脉搏血氧等多种技术方案。通过结构发散，设计了模块化的硬件架构，使得不同功能模块可以灵活组合和升级。

# 市场推广阶段---发散思维：

在市场推广阶段，团队运用发散思维制定了创新的营销策略。他们没有采用传统的广告投放方式，而是通过组合发散，将产品推广与社交媒体、KOL 合作、线下活动等多种渠道结合。通过关系发散，深入研究了目标用户群体的社交网络和兴趣爱好，制定了精准的推广策略。

# 一、发散思维概念

定义: 从一个问题 (信息) 出发, 突破原有的知识圈, 充分发挥想象力, 经不同途径, 以不同的视角去探索, 重组眼前的和记

# 二、发散思维的特点

(1) 流畅性 发散思维 “量” 的指标 流畅性衡量思维发散的速度 (单位时间的量), 是基础。(观念的自由发挥)  
2）变通性 发散思维“质”的指标表现了发散思维的灵活性，是思维发散的关键。（克服僵化的思维框架）  
3）独创性 发散思维的本质，表现发散思维的新奇成分，是思维发散的目的。（新奇反应的能力）

例：法国校长教育淘气的学生。

电影院帽子问题展示了发散思维在日常生活中的应用。印度一家电影院常有戴帽子的妇女观影，帽子挡住了后面观众的视线。如果按照常规思维，可能会发布“禁止戴帽子”的通告，但这可能引起女性观众的不满。

经理运用发散思维，在银幕上打出通告：“本院为照顾衰老有病的女客，允许她们照常戴帽子，在放映电影时不必摘下。”结果所有女客都摘下了帽子。这个解决方案既照顾了特殊群体，又达到了实际效果。

# 三、发散思维三点把握

# 1、正确把握发散思维的特点

保证流畅性；促进灵活性；强调独特性。

# 2、正确看待发散思维的结果

发散思维的结果并不一定都是有价值的，其中的某些结果可能是错误的，但不能因为没有价值就否定发散思维。

# 3、注意对“发散”的把握

尽可能地多方向发散，但不能漫无边际。

# 四、发散思维几种方法

# 1、一般方法

对某一事物的功能、原理、方式、方法等方面为主要内容进行发散，设想其在这些方面的各种可能性。

# 2、假设推测法

假设的问题不是任意选取的，而是有所限定的，所涉及的都应当是与事实相反的情况，是暂时不可能的或是现实不存在的事物对象和状态。

# 3、集体发散思维

发散思维不仅需要用上我们自己的全部大脑，有时候还需要用上我们身边的无限资源，集思广益。例举法，头脑风暴法，设问法，移植法

# 四、发散思维几种方法

# 1.1 功能发散 照明、保暖、保洁

功能发散是以某种功能为扩散点，设想出获得该功能的各种可能性。例如：

照明功能：台灯、蜡烛、手电筒、手机屏幕、荧光棒、镜子反射太阳光、火柴、火把等

保暖功能：晒太阳、电暖器、暖气、暖水袋、摩擦生热、运动、厚衣服等

清洁功能：抹布、扫帚、吸尘器、洗衣机、洗碗机、清洁剂、高压水枪等

# 1.2 用途发散（材料发散） 砖头、曲别针、A4纸

用途发散是以某件物品为发散点，尽可能多地设想它的用途。例如：

砖头的用途：建筑材料（盖房子、铺路、砌墙）、压纸、腌菜、凶器、砝码、哑铃、尺子、多米诺骨牌、画笔（在水泥地上画画）、工艺品等

曲别针的用途：夹文件、做发夹、代替别针、织针、鱼钩、导电材料、开锁工具、艺术品材料等

A4 纸的用途：写字、画画、折纸、垫桌子、包装、扇子、灯罩、收纳盒等

# 四、发散思维几种方法

1.3 结构发散：是以某种事物的结构为发散点，设想出利用该结构的各种可能性。例如：立方体、圆柱体、三角形

- 立方体结构：工具箱、行李箱、收纳盒、收纳凳、包装盒、保鲜盒、香烟盒、火柴盒、车厢、集装箱、船舱、房间等  
- 圆柱体结构：桌子腿、椅子腿、笔杆、笔筒、塔楼、水管、擀面杖等  
- 三角形结构：金字塔、巧克力、三角板、红领巾、屋顶、路标、衣架等

1.4 形态发散：是以事物的形状、颜色、音响、味道、气味、明暗等为发散点，设想出利用某种形态的可能性。例如：红色、铃声、甜味

- 红色的用途：红色指甲油、红色口红、红旗、红墨水、红围巾、红衣服、红色封面笔记本、红色小火车、红十字会标志、红灯笼等  
铃声的用途：学校上课铃、手机闹铃、摇动的铃铛、门铃、自行车铃等  
- 甜味的用途：糖果、蛋糕、蜂蜜、水果、饮料、调味品等

# 四、发散思维几种方法

# 1.5 方法发散

方法发散是以解决问题或制造物品的某种方法为扩散点，设想出利用该方法的各种可能性。例如：吹、敲、拧

- "吹" 的方法: 吹气球、吹灰、吹伤口、吹泡泡、吹口香糖、吹蒲公英、吹笛子、吹蜡烛等  
- "敲" 的方法: 敲桌子、敲门、敲木头、敲书、敲鼓、敲键盘、敲钉子等  
- "拧" 的方法：拧瓶盖、拧螺丝钉、拧毛巾、拧衣服、拧麻花等

1.6 关系发散：是从某一事物出发作为扩散点，尽可能地设想与其他事物的各种联系。例如：玫瑰、人与蛇、网络

玫瑰的关系：爱情、蜜蜂、鲜花饼、情人节、香水、婚礼、刺、美丽等人与蛇的关系：农夫与蛇、蛇皮做乐器、蛇肉可食用、毒蛇咬人、蛇毒制药、宠物蛇等

网络的关系：信息传播、社交、购物、娱乐、学习、工作、游戏、直播等

# 四、发散思维几种方法

# 1.7 因果发散

因果发散是以某个事物发展的结果作为扩散点，推测造成此结果的各种原因；或以某个事物发生的起因作扩散点，推测可能发生的各种结果。例如：

玻璃杯破碎、学习成绩好、如果没有老鼠会发生的结果、如果我是县委书记

- 玻璃杯破碎的原因：生气摔碎、手没拿稳掉地上、热水太烫爆裂、被猫碰到、被弹弓打碎、质量问题等  
·学习成绩好的原因：勤奋努力、学习方法正确、聪明、老师教得好、家庭环境好、有学习兴趣等  
- 如果没有老鼠会发生的结果：减少粮食损失、衣服不会被咬坏、卫生情况变好、不会用老鼠做实验、不会发生鼠疫等

# 2、侧向思维（旁通思维）

从与问题相距很远的事物中受到启发，从而解决问题的思维方式。

当一个人为某一问题苦苦思索时，在大脑里形成了种优势灶，一旦受到其他事物的启发，就很容易与达个优势灶产生相联系的反映，从而解决问题

- 苯环结构的发现：1865 年，德国化学家凯库勒在研究苯分子结构时，因传统链式结构无法解释苯的特殊性质而陷入困境。

一天晚上，他在梦中见到旋转的碳原子长链像一条蛇咬住自己的尾巴，醒来后提出了苯分子是由六个碳原子构成的闭合六元环结构，碳原子间通过交替的单双键连接，这一发现推动了有机化学的发展。

- 基本粒子的能量循环：在现代物理学中，衔尾蛇的形象可用来类比基本粒子的能量循环。

如质子可被看作是一系列电量子涨落与光量子涨落的共振耦合时序模式，粒子不断产生能量又自己消耗能量，以此实现能量在内部的无限循环，就像衔尾蛇自我吞食的过程。

- 元素周期表的循环规律：虽然门捷列夫发现元素周期表的灵感主要来自于纸牌游戏，但从宏观角度看，元素周期表中元素的性质随原子序数的递增呈现周期性变化，这种周期性的循环与衔尾蛇所代表的循环概念有一定的相似性，暗示着化学元素世界中存在的某种内在秩序和循环规律。

# 如何解决等电梯时间过久问题

有一家公司新搬入一幢摩天大楼，不久就遇到了一个难题。由于当初楼内安装的电梯过少，员工上下班时经常要等很长时间，为此怨声不断。于是公司总裁把各部门负责人召集到一起，请大家出谋划策解决电梯不足的问题。经过一番讨论，大家提出了四种解决方案。

第一种：提升电梯上下速度，或者在上下班高峰时段，让电梯只在人多的楼层停。

第二种：各部门上下班时间错开，减少电梯同时使用概率。

第三种：在所有的电梯门口装上镜子。

第四种：安装一部新电梯。

根据爱德华·德·博诺的说法，如果你想出的是第一、第二或第四种，那么你的思维方式属于纵向型或传统型。如果你提出的是第三种，那么你的思维方式属于水平型，属于横向思维。

经过慎重考虑，该公司选择了第三种方案。该方案付诸实施后，员工乘电梯上下，再也没有了抱怨声。爱德华·德·博诺总结说：“等着乘电梯的人一看到镜子，免不了开始观察自己的镜中形象，或者偷偷打量别人的打扮，烦人的等待时刻就在镜前顾盼之间悄悄过去了。该公司的难题固然由电梯不足引起，但也与员工缺乏耐心不无关系。”

# 求同思维

求同思维是指在创造活动中，把两个或两个以上的事物，根据实际的需要，联系在一起进行“求同”思考，寻求它们的结合点，然后从这些结合点中产生新创意的思维活动。

求同思维是从已知的事实或者已知的命题出发，通过沿着单一方向一步步推导，来获得满意的答案以获得客观事物共同本质和规律的基本方法是归纳法，把归纳出的共同本质和规律进行推广的方法是演绎法。这些过程中，肯定性的推断是正面求同，否定性的推断是反面求同。

# 运用求同思维可以按照以下步骤进行:

第一步：在各种不同的场合中找出两个或者两个以上的类似事物。

第二步：寻找这些事物存在的共同特征或联系。

第三步：根据实际需要，从某个“结合点”出发，将这些事物进行“求同”，产生新的创意。

# 新视角求异法

是指对一个事物或问题，要力争从众多的新角度去观察和思考它，以求获得更多的对事物的新认识，萌生和提出更多解决问题的新方法。

是指从解决某一问题的需要出发，思考如何通过采取措施改变事物所包含的要素，从而使事物随之发生符合人的需要的某种变化。

# 问题转换求异法

是指在思考过程中，把不可能办到的问题转换为可以办到的问题，或者把复杂困难的问题转换为简单容易的问题，或者把生疏的问题转换为熟悉的问题，从而找到解决问题的恰当可行或效率更高、效果更好的办法。

# 案例：因果法 -- 追问到底

在日本丰田汽车公司，曾经流行一种管理方法，叫作“追问到底”。也就是说，对公司新近发生的每一件事，都采用追问到底的态度，以便找出最终的原因。一旦找到了最终原因，那就对问题有了深刻的认识。比如，公司的某台机器突然停了，那就沿着这条线索进行一系列的追问。

问: “机器为什么不转了?” 答: “因为保险丝断了。”

问: “为什么保险丝会断?” 答: “因为超负荷而造成电流太大。”

问: “为什么会超负荷?” 答: “因为轴承枯涩不够润滑。”

问: “为什么轴承枯涩不够润滑?” 答: “因为油泵吸不上来润滑油。”

问: “为什么油泵吸不上来润滑油?” 答: “因为抽油泵产生了严重磨。”

问: “为什么油泵会产生严重磨损?” 答: “因为油泵未装过滤器而使铁屑混入。”

追问到此，最终的原因就算找到了。给油泵装上过滤器，再换上保险丝，机器就正常运行了。

# 2、收敛思维一定要有深度

收敛思维时，一定要采取“打破砂锅问到底”的态度，以便找出最终的原因，对一连串的问题就有了深刻的认识，这样收敛思维才有意义

# 3、收敛思维和发散思维的区别

收敛思维与发散思维是一种辩证关系既有区别，又有联系，既对立又统一。

(1) 思维指向相反

收敛思维是由四面八方指向问题的中心，发散思维是由问题的中心指向四面八方。

(2) 两者的作用不同

收敛思维是一种求同思维，为寻求一种最有实际应用价值的结果而把多种想法理顺、筛选、综合、统一。发散思维是一种求异思维，要尽可能地放开，把各种不同的可能性都设想到。

两者结合起来，在充分发散的基础上进行充分的辐合，两者协同动作一个创新过程才能圆满完成

# 收敛思维中的逻辑推理

有一天，某市一家珠宝店发生了一起盗窃案，盗走了价值10万元珠宝，经过公安干警两个多月的侦查，锁定犯罪嫌疑人必是

A、B、C、D 四人中的一人，经审讯，四人提供了下面的口供。

A 说: “珠宝被盗那天, 我在别的城市, 所以我不可能作案”; B 说:

“D是偷盗珠宝的人”; C说:

“B是偷盗犯，三天前我看见他在黑市上卖珠宝”; D说: “B同我有仇, 有意诬陷我, 我不是罪犯”。经过进一步调查取证, 这四人只有一个人说的是真话。你知道犯罪嫌疑人是谁吗? 这四个人中谁说的是真话? ( )

分析：首先 A 的口供里跟 B、C、D 都没关系，先不考虑 A；而 B、D 两人所说的话相互矛盾，因为四人中只有一人说的是真话，所以 B 和 D 的话不可能都是真话，也不可能都是假话，必有一句是正确的。由此利用假设法进行讨论推理。解答：解：（1）假设 B 说的是真话，那 D 就是罪犯，那么：① C 说的是假话，罪犯是 D，不是 B；② D 说的也是假话，他们是没仇，那就是没冤枉他，他就是罪犯；③ A 说的也就是假话，那 A 是罪犯，那 D 是罪犯就不成立；所以此假设不成立。（2）假设 D 说的是真话，那么：① B 说的是假话，D 不是罪犯，②那么 C 说的也是假话，那么 B 不是罪犯，③ A 说的也是假话，所以 A 就在现场，并且作案；故 A 就是罪犯。答：根据上述推理说明 D 说的话是真话，A 是凶手。故选：A。

收敛思维的有效应用需要借助合适的工具和方法：

MECE 法则（Mutually Exclusive, Collectively Exhaustive）是收敛思维的基础工具，用于确保在分析时，各个选项彼此独立且全面覆盖所有可能性。这种方法特别适用于问题分解和方案设计阶段。

问题树通过构建问题的层级结构，帮助明确问题的本质及其各个组成部分。在复杂问题的分析中，问题树能够帮助分析者系统地识别问题的各个维度和关键要素。

决策矩阵通过对不同方案的评估，量化选择标准，辅助决策过程。决策矩阵特别适用于需要考虑多个评估维度的决策场景，如产品选型、供应商选择等。

SWOT 分析评估一个方案的优势、劣势、机会和威胁，帮助做出更加全面的决策。这种方法在战略规划和项目评估中应用广泛。

在工具选择策略上，需要考虑以下因素：

1. 问题的复杂程度：简单问题可以采用基本的逻辑推理方法，复杂问题则需要使用系统化的分析工具。  
2. 时间和资源约束：在时间紧迫的情况下，应选择简单快速的评估方法；在资源充足的情况下，可以采用更加详细的分析工具。  
3. 团队协作需求：如果需要团队协作完成分析，应选择易于理解和操作的工具，确保团队成员能够有效参与。  
4. 结果的精确度要求：对于精确度要求高的决策，应选择定量分析工具；对于方向性决策，可以采用定性分析方法。

# 3.3、线性思维

# 一、什么是线性思维

只用一种眼光、一个角度、一种操作手段去思考、解决问题的思维方法。线性思维是一种直线的、单向的、单维的、缺乏变化的思维方式, 思维沿着一定的线型或类线型 (无论线型还是类线型的既可以是直线也可以是曲线) 的轨迹寻求问题的解决方案的一种思维方法。

静态思维

# 二、线性思维的特点

1、把多元素问题变成一元素问题，一条道走到黑，排除其他道路的可能性  
2、非此即彼，非对即错，不考虑其他方案中正确和有效益的东西，简单地进行二选一的抉择和选择

# 三、线性思维的应用

线性思维虽然有着很强的单一性, 但是在发明创造的过程中遇到问题时, 线性思维又常常是人们思维宝囊中的第一选用对象。

1、当一个问题确定时，人们常常把解决问题的思路径直对准任务，这种思路的产生往往自觉不自觉地导致线性思维的应用，只有在较大的付出之后仍不能解决问题时，人们才去考虑一些其他的思维方法。  
2、由于“线性思维”可以说是构成其他各种思维的基础，我们大家都应该对它有个很好的了解、认识和掌握，并把握好对它的应用，该“单刀直入”的时候，不要犹豫不前，左顾右盼；不该“钻牛角尖”的时候，就果断止步，不要浪费时间、精力。

# 五、线性思维与思维定式

把思维过程用线性来进行描述，目的就是为了使原本抽象的思维过程变得更为形象可感，从而使思路更加清晰，使思维更加开阔，更加灵活，再经过长时间的锻炼，进而培养出更为良好的思维习惯，增强思维能力但同时一定要注意不能陷入思维定式的泥潭。

1、只记相关的词可以节省时间：节约  $50\%$  到  $95\%$  的时间.

5、集中精力于真正的主题。  
6、重要的关键词更为显眼。  
7、重要的关键词并列在时空之中，改善创造力和记忆力。  
8、可以在关键词之间产生清晰合适的联想。

# 3.5、逆向思维

# 一、什么是逆向思维

对司空见惯的似乎已成定论的事物或观点反过来思逆向思维考的一种思维方式。敢于“反其道而思之”，让思维向对立面的方向发展，从问题的相反面深入地进行探索，树立新思想，创立新形象。

如：化学能能产生电能据此意大利科学家伏特 1800 年发明了伏打电池。反过来电能也能产生化学能，通过电解，英国化学家戴维 1807 年发现了钾、钠、钙、镁、锶、钡、硼等七种元素。

# 如何进行逆向思维？

(1) 就事物依存的条件逆向思考,

如：小孩掉进水缸里，把人从水中救起，是使人脱离水，司马光救人是打破缸，使水脱离人。

(2) 就事物发展的过程逆向思考

如：人上楼梯是人走路，而电梯是路走，人不动。

(3) 就事物的位置逆向思考

如：开展假如“我是某某”活动

(4) 就事物的因果关系逆向思考

如：磁场产生电，电场产生磁。
