# 2.2 创新意识

创新意识：指人们根据社会和个体生活发展的需要，引起创造前所未有的事物或观念的动机，并在创造活动中表现出来的意向、愿望和设想。

它是人类意识活动中一种积极的、富有成果性的表现形式,是人们进行创造活动的出发点和内在动力。

# 事例 1: 北极狐的故事

# 奇怪现象

北极狐毛皮每四年会出现一次高峰旅鼠的特点：

繁殖及快，到了繁殖后的第四年，食物便会严重匮乏，大规模迁移，直至最后集体自杀。

艾尔顿 1924 年发表了这一项研究成果, 提出了动物界的食物链理论,并于 1927 年创立了动物生态学。

# 2.3.1 什么是思维定式

思维定式, 也称 “惯性思维”, 是由先前的活动而造成的一种对活动的特殊的心理准备状态, 或是活动的倾向性。

包含：习惯性思维、权威思维、从众思维、自我中心思维、书本型思维等等。

# 习惯性思维：人们不自觉的用某种习惯了的思维方式去思考已经变化的问题。

# 1、行为参照

在许多情境中，人们由于缺乏进行适当行为的知识，必须从其他途径来获得行为引导。根据社会比较理论，在情境不确定的时候，其他人的行为最具有参照价值。

# 2、对偏离的恐惧

对于群体一般状况的偏离，会面临群体的强大压力乃至严厉制裁。研究证明，任何群体都有维持群体一致性的显著倾向和执行机制。对于同群体保持一致的成员，群体的反应是喜欢、接受和优待，对于偏离者，群体则倾向于厌恶、拒绝和制裁。

# 事例 1: 鸟笼逻辑:

甲对乙说: “如果我送你一只鸟笼, 并且挂在你家中最显眼的地方, 我保证你过不了多久就会去买一只鸟回来。”乙不以为然地说: “养只鸟多麻烦啊, 我是不会去做这种傻事的。”

于是，甲就去买了一只漂亮的鸟笼挂在乙的家中。接下来，只要有人看见那只鸟笼，就会问乙：“你的鸟什么时候死的，为什么死了啊？”不管乙怎么解释，客人还是很奇怪，如果不养鸟，挂个鸟笼干什么。最后人们开始怀疑乙的脑子是不是出了问题，乙只好去买了一只鸟放进鸟笼里，这样比无休止地向大家解释要简单得多。

鸟笼逻辑告诉我们，大多数时候人们都受制于强大的惯性思维：鸟笼必定用于养鸟，结婚必先置办新房等。这种惯性思维的益处是，能够帮助我们迅速快捷地认知和适应周围世界。然而，过犹不及，如果把惯性思维扩展到生活的每一个角落，就会成为一种刻板思维。

鸟笼如果设计精巧，其实可以作为观赏品；号称“裸婚”的先结婚后置房，已逐渐为80后所接受；在北欧诸国，由于贫富差距极小，社会公平观念深入人心。所以，不妨偶尔尝试突破鸟笼逻辑，进行发散思维，也许鸟笼之外还有另外一片新天地。

鸟笼逻辑除了可以帮助推销员们推销小鸟和其他商品之外，如果用之适当，还可以成为你生活的小帮手。如培养孩子的阅读兴趣，不要总是喋喋不休地跟他重复读书的重要性。在家里最显眼的地方摆放一些儿童读物，或许会是一种不错的做法。客人拜访的时候，说上句“真是个爱读书的好孩子”，那么羞红了脸的孩子肯定会在客人走后乖乖地端起书本。

PI（Principal Investigator，即能独立带领一个实验室的博导），你拥有所需要的所有资源（优秀的科研团队、充足的经费、完善的实验设备、大把的时间），

“颜宁之问”直指如何培养学生对科学研究甚至未来科学世界的想象力，进而提升学生的创新意识和创造力。

首先，科学的追求在于对未知领域的探索和发现。每一位科学研究者都对未知充满渴望和好奇，这就需要丰富的想象力和创新思维。只有通过大胆的想象，才能打破现有的框架，探索更深入的真理。从这个角度说，科学幻想是推动科学发展的重要动力。

其次，科学幻想（想象力）是进行科学实验的重要工具。科学实验是建立在假设基础上的，这些假设都源自科学家的“幻想”。只有先“幻想”出可能存在的现象，科学家们才能设计出实验去验证，并得出新的科学结论。就像颜宁所说：“不必局限在学术文献里寻找科学问题，也不要总是循着之前博士或博士后实验室的方向、套路走下去……”作为科学研究者就应该有敢于想象、敢于探索的心态。中国科学院院士白春礼在谈及科幻对科学研究者的影响时说，科幻作品很多是在科学前瞻、技术预测基础上产生的，对于社会公众理解科学、相信科学都起到了非常重要的作用。

四、营造“个性化”思维环境，让想象力“自由生长”

# 一、主动打破知识边界，为想象力积累“素材库”

首先，跨学科知识的主动涉猎是基础。

其次，经典与前沿的双向研读能拓宽思维视野。

最后，非学术领域的知识补充能丰富想象力的“维度”。

研究生可利用学校的跨学科选课平台、在线学习资源（如慕课、学术讲座），有针对性地补充与自身研究领域相关的其他学科知识。

# 二、深度参与实践探索，让想象力在“试错”中落地

2.1 在科研实践中“主动提问”，激活想象力  
2.2 在社会调研中“贴近现实”，拓展想象力  
2.3 在创新创业中“大胆试错”，锤炼想象力

例如, 某生物医学领域的研究生在研究 “肿瘤细胞增殖机制” 时, 发现传统实验方法只能观察到细胞的 “静态状态”, 他主动提出 “能否用实时成像技术结合 AI 算法, 动态追踪肿瘤细胞的增殖过程?” ——这个提问打破了 “静态观察”的传统思路, 最终他通过跨学科合作 (结合生物医学与计算机科学), 开发出了 “肿瘤细胞动态增殖监测系统”, 这正是科研实践中 “主动提问”激活的想象力成果。

例如：研究生团队想开发“智能垃圾分类箱”，最初的设计是“通过传感器识别垃圾类型”，但在原型测试中发现“传感器对易腐垃圾的识别准确率低”——他们没有放弃，而是主动调整思路，结合“图像识别技术 + 用户手动辅助分类”，最终提升了识别准确率，产品也获得了市场认可。

# 三、刻意训练思维方式，为想象力搭建“方法论框架”

想象力并非“偶然的灵感迸发”，而是可以通过刻意训练形成的“思维习惯”。对研究生而言，培养“发散思维”“批判性思维”“系统思维”，能为想象力提供稳定的“方法论支撑”，让想象力从“随机闪

现”变为“可持续产生”。例如，在思考如何提升学术论文的创新性”时，尝试列出至少5个方向（如“换一个研究对象”“用新的研究方法”“结合新的理论视角”“关注新的社会现象”“与其他学科交叉”）；在阅读文献时，针对某一研究结论，思考“如果改变研究条件，结论会如何变化？”“除了作者提出的解释，还有其他可能的解释吗？”

例如，在阅读文献时，不仅要理解作者的观点，更要思考“作者的研究方法是否存在缺陷？”“数据来源是否可靠？”“结论是否存在‘以偏概全’的问题？”；在撰写论文时，主动反思“自己的研究假设是否合理？”“有没有遗漏重要的变量？”；在学术讨论中，敢于提出不同意见，比如“我认为这个结论可能需要考虑 XX 因素”“如果从 XX 角度分析，可能会有不同的结果”。

系统思维是想象力的“整合器”，它能帮助研究生从“碎片化思维”转向“整体思维”，在复杂问题中找到“牵一发而动全身”的关键节点。

例如，研究“城市交通拥堵问题”时，不能只关注“增加道路供给”，还要考虑“公共交通优化”“出行需求调控”“智慧交通技术应用”等多个维度的协同；研究“人工智能对就业的影响”时，不仅要分析“岗位替代效应”，还要考虑“新岗位创造效应”“劳动者技能提升需求”“政策监管机制”等整体关联。

例如，某环境科学研究生在研究“河流污染治理”时，通过绘制“河流污染系统图”，梳理出“工业废水排放”“农业面源污染”“居民生活污水”“生态流量不足”等多个要素，并分析了它们之间的“因果关系”——最终他提出了“‘控源截污’生态修复+流域协同管理’的综合治理方案”，这正是系统思维帮助想象力实现“整体突破”的典型案例。

# 四、营造“个性化”思维环境，让想象力“自由生

营造一个“支持想象力生长”的个性化环境，这个环境包括“时间管理”“心态调整”“社交选择”等多个方面，它能帮助研究生减少“思维干扰”，让想象力在“放松且专注”的状态下自然迸发。

例如，在科研中提出的“新假设”，即使最终被实验证明是错误的，也能为后续研究排除“一条错误路径”，甚至从“错误结果”中发现新的问题；在学术讨论中提出的“不成熟观点”，即使被反驳，也能通过交流完善思路，激发新的想象。

可有意识地为自己预留“无目的时间”：比如每天花30分钟散步，不刻意思考科研问题，让大脑自由联想；每周留出半天时间，阅读与专业无关的书籍，或做一些喜欢的爱好（如绘画、摄影），让思维从“学术紧绷”中放松下来。

可主动选择“支持创新”的社交圈：比如加入学校的“跨学科创新社团”，与不同专业的同学交流；参加学术沙龙时，主动与“观点不同”的研究者讨论；在网上关注“领域内有创新思维的学者”，参与他们发起的学术讨论。

想象力的培养并非“遥不可及的目标”，而是“主动破界、深度实践、刻意训练、自由生长”的过程。通过学习，我们了解到：

想象力培养不需要“天赋异禀”，只需要“持续行动”：

需要主动打破知识边界，为想象力积累素材；

需要深度参与实践，让想象力落地；

需要刻意训练思维，为想象力搭建框架；

需要营造个性化环境，让想象力自由生长。

当我们将这些方法融入日常学习与科研中，想象力便会从“偶尔的灵感”，变为“支撑创新的核心能力”，最终让我们在学术道路上走得更远、更稳。
